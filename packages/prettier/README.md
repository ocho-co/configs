# @ocho/prettier-config

Prettier Configuration for OCHO

## Usage

```bash
npm install -D @ocho/prettier-config
```

In prettier config file:

```js
// .prettierrc.js
import config from '@ocho/prettier-config'

export default config
```

Or in `package.json`:

```json
{
  "prettier": "@ocho/prettier-config"
}
```
